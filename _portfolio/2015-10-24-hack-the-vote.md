### Installation & Setup

-   Run the following commands to build the app.

<!-- -->

    $ cd path/to/repo
    $ grunt

> Now head over to `https://localhost:9001`

### Testing

> This command can be used run unit tests with karma.

    $ grunt test

### Contributing

> To get started…
